# Prompt Templates

## 模板 A：通用长文转思维导图

你是专业思维导图专家。请将我提供的内容整理为“可直接导入/粘贴到 XMind 的 Markdown 思维导图结构”。

要求：
- 输出仅为 Markdown，不要解释。
- 第一行使用 `#` 作为总主题。
- 用嵌套 `-` 表示层级。
- 一级分支控制在 5-9 个。
- 同层级保持并列、对称。
- 保留关键数字、公式、阈值、阶段。
- 删除重复表达与宣传性口号。
- 每个节点尽量短句（不超过 18 个字，必要时可稍长）。
- 结尾增加“行动清单（3-7条）”。

输入内容：
{{INPUT_TEXT}}

## 模板 B：实操教程转 SOP 思维导图

你是“电商/SOP 架构型思维导图”专家。请把输入教程重构为可执行 SOP 的 Markdown 思维导图。

要求：
- 输出仅 Markdown。
- 结构必须包含：
  1) 目标与适用范围
  2) 核心指标与公式
  3) 分阶段流程
  4) 决策分支（If/Then）
  5) 风险与合规
  6) 每日/每周执行清单
- 对时间周期、阈值、入口路径、工具名称单独建子节点。
- 将“概念描述”改写为“动作指令”。
- 标注高风险动作为 `⚠️`。

输入内容：
{{INPUT_TEXT}}
