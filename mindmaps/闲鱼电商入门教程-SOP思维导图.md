# 闲鱼电商入门教程：SOP思维导图

- 目标与适用范围
  - 目标
    - 冷启动起店
    - 稳定出单
    - 放大爆款
  - 适用
    - 实物/虚拟商品
    - 新手到进阶卖家
- 核心指标与公式
  - 成交率
    - 成交率=咨询/成交
  - 虚拟转化参考
    - 比值=浏览/想要
    - 越接近1越好
  - 投放指标
    - 转化率
    - CPC=花费/点击
    - 点击率=点击/曝光
- 启动流程（0-7天）
  - 货源
    - 1688/拼多多源头工厂
    - 可无痕发货商家
  - 养号（三天）
    - 第1天：搜索+咨询同行
    - 第2天：完善资料+认证
    - 第3天：上架测流量
  - 首批链接
    - 复制爆款标题+主图
    - 价格略优先拿数据
- 起量流程（7-15天）
  - 选潜力链接
    - 72小时内自然出单优先
    - 或曝光/浏览/想要良好
  - 付费工具
    - 优推抵扣
    - 急速卖（优先套餐1/2）
    - 超级擦亮
  - 决策
    - 有曝光无成交→补单+投放
    - 数据差且无人问→下架
- 放大流程（15-30天）
  - 流量类型
    - 曝光量：解决被看见
    - 浏览量：解决进店点击
    - 关注收藏：矫正人群标签
    - 询单量：高客单临门一脚
  - 老爆款维护
    - 自然流下滑时小幅提价竞价
    - 观察投产比决定去留
- 决策分支（If/Then）
  - If 搜索占比高且不开单
    - Then 补单预热+浏览/询单投放
  - If 首页占比高且流量不准
    - Then 老链补单+关注收藏矫正
  - If 30天后钱花不出去或无转化
    - Then 下架清理
- 店铺运营与矩阵
  - 链接生命周期
    - 7天看潜力
    - 15天看竞争力
    - 30天看长期价值
  - 矩阵规则
    - 一爆品多城市差异化
    - 不同账号不同图文标题价格
- 风险与合规
  - ⚠️ 刷单/补单存在平台风险
  - ⚠️ 高仿赛道存在封号与法律风险
  - ⚠️ 差评处理避免违规取证与隐私问题
- 每日/每周执行清单
  - 每日
    - 看七天来源分布
    - 调整1-2个在投链接
    - 回复咨询并提效话术
  - 每周
    - 清理死链接
    - 复盘ROI与流量结构
    - 更新爆款对标素材库
